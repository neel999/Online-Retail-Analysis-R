# Online Retail Sales Analysis & Dashboard

تحليل مبيعات متجر أونلاين باستخدام R + tidyverse + flexdashboard

## المشروع يغطي:
- تنظيف البيانات (إزالة الإلغاءات، القيم السالبة، NA)
- حساب إجمالي المبيعات، أفضل المنتجات، الاتجاهات الشهرية، توزيع المبيعات حسب البلد واليوم
- RFM Analysis للعملاء
- داشبورد تفاعلي

## النتائج الرئيسية:
- إجمالي الإيرادات: ≈ £8.9 مليون
- أكثر من 18 ألف طلب من 4,338 عميل
- أعلى مبيعات في المملكة المتحدة (بفارق كبير)
- موسمية واضحة (ارتفاع في نوفمبر-ديسمبر)

## الأدوات المستخدمة:
- R + tidyverse (dplyr, tidyr, ggplot2)
- janitor, lubridate, scales, flexdashboard, DT
